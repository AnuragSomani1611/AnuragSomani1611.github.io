PhonicVision

An application based project built to help the impaired people by reading text content from images. The application uses OCR technology API by google and developed with Flutter and Dart. The application first recognizes the text from the image and then it uses the text to generate audio and provide its pdf form for user to downlaod it. The application also provides the user with this service when there is no internet available.

Gesture Based Tool for Sterile Browsing of Radiology Images

In this duration I designed various deep learning algorithms, Flask development and IBM cloud services. During externship I worked with a team and build a "Gesture based tool for Sterile Browsing of Radiology Images" using CNN, OpenCV and Flask. MITM Detector

Its a small python based software tool to detect man in the middle attack. In the project a simple python program is written to store initial MAC address from ARP table and then to save the new address from the ARP table, and if the initial and final address doesn't mmatch then it will disrupt the signal and break the connection showing the network connection status as bad.
